## Tips for Using the AI Scheduler

1. **Be Specific**: Include exact time slots and day preferences
2. **Clear Constraints**: Specify work hours, commute time, break requirements
3. **Priority Ranking**: Mention which courses are more flexible than others
4. **Special Requirements**: Include any health, transportation, or personal needs
5. **Format Consistency**: Use consistent formatting for courses and times
